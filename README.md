#  Tour Planner

An intelligent tour management and booking platform designed to modernize travel planning through a structured, scalable, and user-centric system.

---

##  Overview
Tour Planner is a full-stack web application that transforms manual tour operations into a digital platform. It provides seamless interaction between users and administrators, enabling efficient tour management, booking workflows, and data handling.

---

## Tech Stack
- Python (Flask)
- HTML, CSS, Bootstrap
- SQLite
- Jinja2 (Template Engine)

---

##  Core Features

###  User Module
- Dynamic tour browsing with structured listings
- Detailed tour pages with pricing, itinerary, and availability
- Smart booking system with form validation
- Real-time feedback on booking status
- Responsive UI for cross-device accessibility

---

###  Admin Module
- Full CRUD operations for tour packages
- Centralized dashboard for managing tours and bookings
- Data-driven control of pricing, availability, and schedules
- Efficient booking management system

---

###  System Features
- Modular Flask architecture (separation of concerns)
- Template inheritance for maintainable UI
- Lightweight relational database (SQLite)
- Clean routing structure for scalability
- Organized static asset management

---

## Highlights
- Designed with scalability in mind
- Follows MVC-like architecture pattern
- Optimized for simplicity and performance
- Suitable for small-scale travel businesses transitioning to digital platforms
